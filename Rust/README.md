# Rust

- Create new project
> cargo new helloworld

- Compile [on Windows]
> rustc src\main.rs

- Run project [on Windows]
> .\main

## Cargo
- Check version
> cargo --version

- Create new project
> cargo new hello_cargo --vcs=git

- Create new project with .gitignore
> cargo new hello_cargo --vcs=git

> cd hello_cargo

- Build Project
> cargo build

- Build with release
> cargo build -r

- Run project
> cargo run
